GitHub & Git

For this step, I first needed to create a GitHub account in order to be able create a repository to store my files. Since I have Linux installed on my Virtualbox, I use the terminal console on my Linux and enter the following codes to setup my GitHub repository:
git init
git config --global user.name "Chi Li"
git config --global user.email "cli16@mail.wou.edu"
git remote add origin https://github.com/cli16/cli16.github.io.git

Then, When I am ready to publish my files onto GitHub I would enter the following Git commands:
git add
git commit -m "message"
git status
git push origin master

Webpage

Now, the most funnest part of this assignment is building the webpages.

As I've said before, I have no experiences with bootstrap4, so I did a bit of research on the internet and learn how to implement it into my webpages. From the W3Schools's Bootstrap Tutorial, I learned that I can either have to download the bluestrap and host it myself or I can use the Bootstrap CDN (easiest way).

As you might guessed it, I chose the easiest way and implemented bluestrap4 into my webpages usig Bootstrap CDN (Content Delivery Network).
<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.7/css/bootstrap.min.css">
<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.3.1/jquery.min.js"></script>
<script src="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.7/js/bootstrap.min.js"></script>

Note that those CDN links must be placed inside head tag.
